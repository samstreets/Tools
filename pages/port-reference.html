Port Ranges Guide

Well-Known Ports (0–1023)

Reserved for system and widely-used services. Require root/admin privileges to bind. Assigned by IANA and universally recognised.

Registered Ports (1024–49151)

Registered by software vendors with IANA. Don't require elevated privileges. Used by many popular applications and services.

Dynamic / Ephemeral (49152–65535)

Assigned automatically by the OS for outbound client connections. Not permanently assigned — used temporarily for a session's duration.

TCP vs UDP

TCP ensures reliable, ordered delivery — ideal for web, email, and file transfers. UDP is faster but connectionless — used for streaming, DNS, and VoIP.
